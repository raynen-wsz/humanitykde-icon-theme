Humanity Icon Theme for KDE Plasma.

### Overview ####
Humanity KDE Icon Theme is a custom icon theme designed specifically for KDE Plasma. All icons are created in SVG format. Some icons were updated and recolored, while others were created from scratch using Gnome/Humanity templates.

### Installation ###
Copy the Humanity KDE folder to your icon themes directory:

For a single user: ~/.local/share/icons/
For system-wide: /usr/share/icons/
Apply and enjoy the refreshed icon theme.

### Notes ###
This icon theme may also work on other environments, but it hasnt been tested.
