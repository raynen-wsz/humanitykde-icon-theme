Humanity KDE Icon Theme

Version: 0.1.0
Date: December 2025
Author: ranen

### Overview ####
Humanity KDE Icon Theme is a custom icon theme designed specifically for KDE Plasma. All icons are 100% created in SVG format, also some icons were updated and recolorized
Around 14% of the icons have been replaced.

### Installation ###
Copy the Humanity KDE folder to your icon themes directory:

For a single user: ~/.local/share/icons/
For system-wide: /usr/share/icons/
Apply and enjoy the refreshed icon theme.
